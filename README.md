⚡ Ray-Ban Modern Landing Page

A visually stunning, interactive landing page inspired by the Ray-Ban aesthetic.
This project utilizes advanced WedbGL -based image effects and smooth typography
transitions to create a premium user experience.

✨ Features

. WebGL Image Transitions: Powered by Shery.js (Style 5) for a liquid-like distortion effect.

. Synced Typography: Text changes seamlessly with the background image using custom GSAP logic.

. Custom Cursor: Interactive mouse-following effects.

🛠️Tech Stack

. HTML5 & CSS3(Custom Layouts & Keyframes)

. JavaScript(ES6+)

. GSAP (GreenSock Animation Platform)

. Shery.js (For WebGL Effects)

. Three.js (Backend for WebGL rendering
